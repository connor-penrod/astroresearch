SAOImageDS9 Version 1.5 Release Notes

    Version 1.5

    1. Fixed problem with Windows style Open file dialog box
    2. Now supports following coords systems:
      • Image (was Logical)
      • CCD (was Physical)
      • Detector (was Mosaic Physical)
      • Amplifier
      • WCS
    3. New version of Doug Mink's WCS libs
    4. Fixed minor problems with interactive Pan Mode
    5. Added "Source TCL" file support
    6. Open Mosaic Image command

      7. Will open multiple fits files as a mosaic.
      Supports following patterns:
      *#.fits
      *_#.fits
      *.IM*.fits
    7. Open Mosaic Segment command
    8. Fixed minor problem with zscale and high energy data
    9. The Magnifier now Works!

      10. There are two algorithms available (under Preferences)
      Magnification can be selected via preferences
      1. Fast (and less Accurate)
      2. More Accurate (and slower)
    10. Improvements to Dialogs
    11. Now supports (unoffical)FITS data size -16 (unsigned short)
    12. Support for byte swapped processors
    13. Support large images > 8k x 8k
    14. Loading images are now 10x faster
    15. Supports BLANK Fits keyword
    16.  Menu Short Cuts have been added (use Alt key, such as Alt-f for Frame menu)
    17. Horizontal and Verticle Graphs now work in all modes (not just Crosshair)
    18. ds9 will accept fits files via STDIN (ie. $ cat foo.fits | ds9 -file -  )
    19. command line argument support (ie. $ds9 -? for list)
    20. crosshair position is more accurate
    21. added new scales: 'ln' and 'log'.
      1. ln --  SAOimage natural log based algorithm
      2. log--  IRAF log base 10 algorithm
    22. Support for Postscript printing (Level 1 and Level 2)
    23. Blinking is now supported
